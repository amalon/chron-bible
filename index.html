The Bible in Chron

This project aims to represent chronological and genealogical information from the Bible using chron, a software project designed for representing such data using the Prolog logic programming language.

The aim is to be able to automatically derive chronological timelines and genealogical graphs from it, with references back to the source Bible verses.

You can view some of this data online using chronweb, though currently it is limited to genealogical data.
